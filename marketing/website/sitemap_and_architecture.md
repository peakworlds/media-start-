# Sitemap & Architecture: Peak Haven Australia

## URL Structure
- `/` [Homepage]: Hero Video, 10x Guarantee, Core Protocols grid.
- `/catalogue` [Shop]: Grid view of individual diagnostic vials.
- `/protocols` [Stacks]: Dedicated landing pages for multi-vial arrays (Metabolic, Cellular, Fibroblast).
- `/verification` [Testing]: Embed of Janoshik HPLC / Mass Spectrometry PDFs cross-indexed by batch number.
- `/haven` [Community]: Password protected or redirect to Telegram landing page.
- `/support` [Contact]: Stark ticketing system. Explicit terms and conditions.

## Architectural Vibe
The site must operate like a headless Shopify instance so it never "feels" like a standard theme. The geometry must be sharp (no rounded buttons), massive high-contrast typography, and a distinct lack of "lifestyle photography." It should look like a portal to a multi-billion dollar laboratory server.
